# 示例代码总览

官方示例已经按语言拆分成独立页面，适合直接检索、收藏和跳转。

## 使用建议

- 第一次接触接口时，先看对应语言总览页
- 确认接口含义后，再进入单个示例页复制代码
- 需要对照实现时，可在 Python 与 C++ 两套示例之间横向切换
