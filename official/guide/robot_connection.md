# 2.2 机器人手机APP连接（Agibot Go APP）

最新 APP 版本号需与机器人最新固件版本对应，请下载正确版本使用。

* * *

<a id="id1"></a>

## 2.2.1 下载地址

<a id="ios"></a>

### iOS

  * App Store 搜索应用 _Agibot Go APP_

  * 扫描下方二维码下载 Agibot Go APP

![iOS 下载二维码](/official/_images/ios_qr.png)

<a id="android"></a>

### Android

  * 安卓应用市场搜索应用 _Agibot Go APP_

  * 扫描下方二维码下载 Agibot Go APP

![Android 下载二维码](/official/_images/ios_qr.png)

<a id="id2"></a>

### 操作步骤

<a id="id3"></a>

#### 添加机器人

  1. 打开“添加机器人”界面。

<a id="id5"></a>

![打开添加界面](/official/_images/add_robot_1.png)

打开添加界面

  2. 添加机器人。

<a id="id6"></a>

![添加机器人](/official/_images/add_robot_2.png)

添加机器人

* * *

<a id="wi-fi"></a>

#### Wi-Fi 连接机器人

  1. 点击 **Wi-Fi 连接** 。

<a id="id7"></a>

![Wi-Fi 连接](/official/_images/wifi_1.png)

Wi-Fi 连接

  2. 输入 Wi-Fi 密码。

<a id="id8"></a>

![输入 Wi-Fi 密码](/official/_images/wifi_2.png)

输入 Wi-Fi 密码

* * *

<a id="ap"></a>

#### AP 热点连接机器人

  1. 点击 **AP 热点连接** 。

<a id="id9"></a>

![AP 连接](/official/_images/ap_1.png)

AP 连接

  2. 选择连接。

<a id="id10"></a>

![选择连接](/official/_images/ap_2.png)

选择连接

* * *

<a id="id4"></a>

#### 连接结果通知

  1. 连接成功。

<a id="id11"></a>

![连接成功](/official/_images/result_1.png)

连接成功

  2. 可正常交互。

<a id="id12"></a>

![可正常交互](/official/_images/result_2.png)

可正常交互
