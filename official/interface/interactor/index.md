# 5.2 交互模块

**智元机器人X2 AimDK交互模块 - 提供丰富的人机交互接口**

**核心功能**

  * **语音交互** ：语音识别、语音合成、语音命令处理

  * **表情控制** ：丰富的表情显示和情感表达

  * **多媒体播放** ：视频、音频、图像播放能力

  * **触觉反馈** ：触觉传感器和反馈控制

**接口规范**

  * **服务接口** ：大部分采用`/aimdk_5Fmsgs/srv/`和`/face_ui_proxy/`前缀

  * **消息类型** ：采用`aimdk_msgs`包

  * **编程语言支持** ：C++、Python

  * **消息格式** ：标准ROS2消息格式

**版本兼容性**

  * **当前版本** : v0.8

  * **最低版本** : v0.6 (部分功能)

  * **ROS2版本** : Humble

  * **支持架构** : x86_64, aarch64

**安全注意事项**

警告

**重要安全提醒**

  * 语音合成服务有优先级限制，避免同时播放多个语音

  * 表情播放会占用显示资源，注意资源管理

  * 多媒体播放需要确保文件路径正确且可访问

小心

ROS的服务(service)机制在跨板通信时存在一些待优化问题, **二次开发时请参考例程添加异常处理、快速重试等保护机制**
