# 1.2 机器人具体参数

**AgiBot X2 系列** 具备完善的 **硬件与系统资源** ，包括**机械结构** 、**传感器系统** 、**通信接口** 、**计算单元与电源系统** 。依托这一高度集成的架构，灵犀X2能够实现 **复杂的全身运动控制** ，**智能感知与环境交互** ，并通过**AimDK** 支持 **高级二次开发** 。
