# 1.1 产品简介

<a id="x2-ultra"></a>

## 1.1.1 旗舰版（X2 Ultra）

  * 旗舰版整体具备 **30 个自由度** ，其中头部具备 **1 个自由度** ，单手臂具备 **7 个自由度** ，包括肩身关节、上臂关节、手肘关节和手腕关节。  
单腿具备 **6 个自由度** ，包括胯关节、腿关节、膝关节和踝关节。腰部具备 **3 个自由度** （旋转、俯仰、侧摆）。

  * 旗舰版标配 **激光雷达、前视双目 RGB 摄像头、RGBD 相机与后视 RGB 摄像头** ，  
并配备高算力板 **Nvidia Orin NX** 计算单元，标配 **4G/5G 模块** 。  
支持选装 **OmniPicker** 与 **OmniHand** ，并可选配遥操作配件及充电站配件。

  * 该版本具备全量的二次开发能力，同时支持灵创平台使用。
