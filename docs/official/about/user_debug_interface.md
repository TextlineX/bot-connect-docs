# 1.4 用户调试接口（旗舰版）

序号 | 接口类型 | 接口简称 | 接口说明  
---|---|---|---  
**1** | RK3588 USB | USB Type-A | 支持 USB3.0 host，5V/1.5A 电源输出，RK3588 USB Type-A 与 Type-C 接口共享 5Gbps 带宽  
**2** | RK3588 USB | USB Type-C | 支持 USB3.0 host，5V/1.5A 电源输出，RK3588 USB Type-A 与 Type-C 接口共享 5Gbps 带宽  
**3** | Orin NX USB | USB Type-A | 支持 USB3.0 host，5V/1.5A 电源输出，Orin NX USB Type-A 与 Type-C 接口共享 10Gbps 带宽  
**4** | Orin NX USB | USB Type-C | 支持 USB3.0 host，5V/1.5A 电源输出，Orin NX USB Type-A 与 Type-C 接口共享 10Gbps 带宽  
**5** | SIM 卡座 | SIM 卡座 | 5G 模组 Sim 卡座
