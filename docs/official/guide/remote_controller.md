# 2.3 遥控手柄连接

备注

遥控手柄在出厂时已默认连接，无需重新配对。使用时，只需按下遥控器中央 PS 5 logo 按键即可快速建立连接。

* * *

<a id="id2"></a>

## 2.3.1 操作步骤

**1\. 打开“添加机器人”界面**

<a id="id3"></a>

![进入蓝牙连接界面](/official/_images/step_1_r.png)

进入蓝牙连接界面

**2\. 连接机器人后，通过机器人设置 - > 蓝牙 -> 选择手柄来进行连接**

<a id="id4"></a>

![点击蓝牙设置](/official/_images/step_2_r.png)

点击蓝牙设置

**3\. 选择手柄并点击进行连接**

<a id="id5"></a>

![选择手柄并点击进行连接](/official/_images/step_3_r.png)

选择手柄并点击进行连接
